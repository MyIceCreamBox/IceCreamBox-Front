## 🍡 Git Commit Message

| 헤더     | 내용                                                |
| -------- | --------------------------------------------------- |
| feat     | 새로운 기능 추가                                    |
| refactor | 코드 리팩토링                                       |
| fix      | 버그 수정                                           |
| style    | 세미 콜론 수정, 줄바꿈, css 변경 등 기능과 무관한 스타일 변경 |
| setting  | 빌드수행, 패키지 설치, 환경 설정 수정 등            |
| docs     | 주석 추가 및 삭제, readme 작성                         |
